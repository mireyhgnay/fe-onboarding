**💡 필요한 지식**
- npm 프로젝트 구축과 웹팩 설정
- DOM 조작, 생성
    - dom 객체 가져오기: querySelector, getElerhkmentById
    - 속성 지정: setAttribute 또는 직접 지정
    - dom 만들기: createElement
    - dom 추가: append
- 이벤트
    - 클릭: onclick 또는 addEventListener로 click 추가

**👩🏻‍💻 npm start**
```bash
$ cd study

$ npm start    // build
```
